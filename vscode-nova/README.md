# Nova Programming Language Support (Nova Language)

[Tiếng Việt](#tiếng-viet) | [English](#english)

---

## Tiếng Việt

**Nova Programming Language Support** là extension chính thức dành cho trình soạn thảo Visual Studio Code, cung cấp công cụ hỗ trợ phát triển các ứng dụng sử dụng ngôn ngữ lập trình **Nova**.

### Tính năng nổi bật

* **Tô màu cú pháp (Syntax Highlighting)**: Hỗ trợ đầy đủ các từ khóa, kiểu dữ liệu, cấu trúc điều khiển, chuỗi văn bản, chú thích và hằng số của ngôn ngữ Nova (bao gồm cả cú pháp tiếng Việt và tiếng Anh).
* **Cấu hình ngôn ngữ (Language Configuration)**: Tự động đóng ngoặc, thụt lề thông minh, và hỗ trợ các ký tự comment (`//`, `/* */`).
* **Snippets tiện lợi**: Cung cấp các mẫu mã nguồn viết nhanh cho các cấu trúc cơ bản (`hàm main`, `vòng lặp`, `khai báo biến`, v.v.).
* **Chạy mã trực tiếp (Run Current File)**: Nhấp chuột phải vào trình soạn thảo và chọn **"Nova: Run Current File"** để chạy tệp `.nova` hiện tại của bạn một cách nhanh chóng.

### Yêu cầu hệ thống

Để chạy được mã nguồn Nova, bạn cần cài đặt **Nova Programming Language** trên máy tính thông qua gói cài đặt chính thức `Nova-Setup.exe` hoặc tự cấu hình trình biên dịch/thông dịch trong biến môi trường `PATH`.

## English

**Nova Programming Language Support** is the official Visual Studio Code extension providing developer tooling and rich language support for the **Nova** programming language.

### Key Features

* **Syntax Highlighting**: Comprehensive syntax coloring for keywords, data types, control flow structures, string literals, comments, and constants in Nova (supporting both Vietnamese and English syntax).
* **Language Configuration**: Smart auto-closing brackets, intelligent indentation, and comment toggling (`//`, `/* */`).
* **Code Snippets**: Quick templates for common boilerplate code (`main function`, `loops`, `variable declarations`, etc.).
* **Run Current File**: Right-click anywhere in your editor and select **"Nova: Run Current File"** to quickly execute the current `.nova` file.

### Requirements

To execute Nova source files, you must have the **Nova Programming Language** installed on your system via the official installer `Nova-Setup.exe` or have the compiler/interpreter configured in your system's `PATH` environment variable.
