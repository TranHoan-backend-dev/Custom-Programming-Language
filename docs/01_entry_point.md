# ⚙️ Cú pháp khởi chạy (Entry Point)

[« Quay lại README](../README.md)

---

Mọi chương trình bắt đầu từ hàm `main`. Đây là điểm khởi đầu (linh hồn) của ứng dụng.

### Cú pháp

```text
// Phiên bản tiếng Anh (English version)
main() -> void {
    // Code của bạn ở đây
}

// Phiên bản tiếng Việt (Vietnamese version)
main() -> trống {
    // Mã chương trình ở đây
}
```
